## 🚀 立即运行

### 1. 编译项目

```bash
./gradlew clean

```

```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```

## 🎨 添加其他模型

### 模板步骤

```
1. 复制 StyleClassifierExecutor.java
   └─> 重命名为 YourModelExecutor.java
   └─> 修改类名和逻辑

2. 复制 StyleClassifierFragment.java
   └─> 重命名为 YourModelFragment.java
   └─> 修改 UI 和交互

3. 在 FragmentFactory 中注册
   └─> 添加 getYourModelFragment() 方法

4. 在 MainActivity 中添加
   └─> fragments.add(FragmentFactory.getInstance().getYourModelFragment())

5. 配置 Tab 资源
   └─> strings.xml: 添加标题
   └─> arrays.xml: 添加图标
```

## ⚠️ 注意事项

### 必须检查
- [ ] MindSpore Lite 库已正确引入
- [ ] 模型文件格式为 .ms
- [ ] 设备为 ARM 架构（不支持 x86 模拟器）
- [ ] 输入尺寸与模型匹配（当前为 224x224）
- [ ] 预处理参数与训练时一致

### 性能提示
- 使用 2-4 个线程
- 在后台线程执行推理
- 及时释放资源
- 考虑模型量化

### 常见错误

**模型加载失败：**
- 检查文件路径
- 确认文件格式
- 验证 MindSpore 版本

**推理崩溃：**
- 检查输入尺寸
- 验证数据类型
- 确保内存充足

**UI 不显示：**
- 检查 Fragment 注册
- 验证 Tab 配置
- 查看布局文件
