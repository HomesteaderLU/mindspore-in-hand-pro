## 📊 技术栈

### Android 端
- **语言**：Java
- **最小 SDK**：21 (Android 5.0)

- **UI 框架**：Material Design
- **AI 引擎**：MindSpore Lite（可选）
- **架构模式**：MVP + Fragment

- **语言**：Python 3.7+
- **深度学习框架**：MindSpore 2.0
- **图像处理**：Pillow

## ⚙️ 配置参数

### 训练参数（可调整）

```python
NUM_CLASSES = 10          # 类别数量
IMAGE_SIZE = 224          # 输入图像尺寸
BATCH_SIZE = 32           # 批次大小
EPOCHS = 50               # 训练轮数
LEARNING_RATE = 0.001     # 学习率
NUM_THREADS = 4           # CPU 线程数
```

### 模型性能

| 指标 | 数值 |
|------|------|
| 模型大小 | ~15 MB (FP32) / ~4 MB (INT8) |
| 推理时间 | 50-200ms（取决于设备） |
| 准确率 | 80-85%（取决于数据质量） |
| 内存占用 | <100 MB |

## 🐛 已知问题

### 限制
1. ⚠️ 仅支持 ARM 架构设备
2. ⚠️ x86 模拟器无法运行
3. ⚠️ 需要至少 2GB 可用内存
4. ⚠️ 首次加载模型较慢（~2秒）

###  workaround
- 使用真机测试
- 关闭其他应用释放内存
- 模型缓存到本地避免重复加载
