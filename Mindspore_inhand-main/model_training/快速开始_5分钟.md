## 📋 分步指南（如果想手动操作）

### 步骤 1：安装 Python 依赖

```bash
cd model_training
pip install -r requirements.txt
```

### 步骤 2：准备数据集

**选项 A：使用自己的图片**

创建以下目录结构：

```
raw_images/
├── oil_painting/      # 放油画图片
├── watercolor/        # 放水彩图片
├── sketch/            # 放素描图片
└── ... (其他 7 个类别)
```

运行：
```bash
python prepare_dataset.py
```

**选项 B：跳过数据集（使用模拟数据）**

直接跳到步骤 4

### 步骤 3：训练模型

```bash
python train_style_classifier.py
```

等待训练完成（约 30-60 分钟，取决于硬件）

输出文件：
- `checkpoints/` - 训练检查点
- `style_classifier.air` - AIR 格式模型

### 步骤 4：转换为 MS 格式

```bash
python convert_model.py \
    --input style_classifier.air \
    --output style_classifier.ms \
    --android-path ../
```

这会自动将模型复制到 Android 项目的 assets 目录。

### 步骤 5：启用真实推理

## 🐛 遇到问题？

### 问题 1：编译失败

**错误**：`Could not find mindsporelibrary`

**解决**：
```bash
# 确保 mindsporelibrary 模块存在
ls mindsporelibrary/

# 如果不存在，从原项目复制
```

### 问题 2：模型加载失败

**错误**：`Failed to load model`

**检查**：
1. 模型文件是否在 assets 目录
2. 文件格式是否为 .ms
3. 设备是否为 ARM 架构

**调试**：
```bash
adb shell ls /data/data/com.mindspore.himindspore/cache/
```

### 问题 3：应用崩溃

**查看日志**：
```bash
adb logcat -s StyleClassifier:E
```

**常见原因**：
- 空指针异常 → 检查模型是否正确加载
- 内存不足 → 减小图片尺寸
- 权限问题 → 检查 AndroidManifest.xml
