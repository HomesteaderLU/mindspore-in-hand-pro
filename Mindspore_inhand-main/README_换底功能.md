## 1️⃣ 背景替换

### 功能描述

基于华为 HMS ML Kit 的**实时人像分割**能力，将摄像头中的人物从原背景中分离出来，替换为选择的背景图片。

### 使用步骤

1. 在 **体验 → 高级区** 点击 **"背景替换"**
2. 允许相机权限
3. 摄像头实时显示人像分割效果
4. 从底部横向列表选择预设背景图
5. 点击末尾 **"+"** 按钮可自定义背景图片
6. 点击右上角 **保存** 按钮保存结果

### 技术原理

```
摄像头画面 → [HMS ML Kit 人像分割] → 提取前景人像
    ↓
前景人像 + 背景图片 → [合成] → 换底结果
```

### 相关文件

| 文件 | 说明 |
|------|------|
| `hms/.../PortraitSegmentationActivity.java` | 主 Activity，集成相机预览和分割 |
| `hms/.../PortraitSegmentationAdapter.java` | 底部背景选择列表适配器 |
| `hms/.../OnBackgroundImageListener.java` | 背景选择回调接口 |
| `hms/.../res/layout/activity_portrait.xml` | 页面布局 |

## 2️⃣ 证件照换底色

### 功能描述

选择一张人像照片，自动分割出人物主体，然后合成到**纯红 / 纯白 / 纯蓝**背景上，生成标准证件照。

### 使用步骤

1. 在 **体验 → 高级区** 点击 **"证件照换底色"**
2. 点击 **"图片"** 从相册选择人像照片，或点击 **"相机"** 拍照
3. 等待自动处理完成（上方显示原图，下方显示结果）
4. 点击 **红底 / 白底 / 蓝底** 按钮切换背景色
5. 点击 **"保存"** 将结果保存到相册

### 技术原理

```
选择图片 → [HMS ML Kit 人像分割] → 提取前景人像（透明背景）
    ↓
透明人像 + 纯色背景 → [Canvas.drawColor + drawBitmap] → 证件照
    ↓
切换颜色 → 重新合成 → 实时预览
    ↓
保存 → [Bitmap 合成] → 存入相册
```

### 支持的底色

| 颜色 | 色值 | 按钮 |
|------|------|------|
| 🔴 **红底** | `#C0362C` | 红底 |
| ⚪ **白底** | `#FFFFFF` | 白底 |
| 🔵 **蓝底** | `#4A6FF5` | 蓝底 |

## 🔍 常见问题

### Q1: 为什么需要华为 HMS Core？

**A**: 两个功能均基于华为 ML Kit 的人像分割能力，需要在华为设备或安装了 HMS Core 的设备上运行。

### Q2: 分割不准确怎么办？

**A**: 
- 确保光线充足
- 人物与背景颜色差异大时效果更好
- 避免复杂背景（纯色背景效果最佳）
- 避免多人同时出现在画面中

### Q3: 保存的图片在哪里？

**A**: 保存在系统相册中，可在相册 App 中查看。

### Q4: 能否添加更多底色？

**A**: 可以。在 `IdPhotoActivity.java` 中添加新的颜色常量，并在布局中增加对应按钮即可。
